# Biblioteca-universitaria-
Applicazione desktop per la gestione di una biblioteca universitaria, sviluppata in JavaFX con Scene Builder.

Il progetto è stato realizzato per l’esame di Ingegneria del Software, con particolare attenzione alla progettazione architetturale, alla separazione delle responsabilità e al rispetto delle buone pratiche di sviluppo.

🏗 Architettura

Struttura basata su pattern MVC (Model-View-Controller)

Separazione tra logica di business, interfaccia grafica e gestione dati

Interfacce sviluppate in FXML con controller dedicati

🎓 Approccio

Il progetto ha previsto:

Analisi dei requisiti

Modellazione UML (diagrammi di classe e casi d’uso)

Progettazione modulare

Applicazione di principi OOP (incapsulamento, coesione, basso accoppiamento)

Validazione e gestione degli errori
